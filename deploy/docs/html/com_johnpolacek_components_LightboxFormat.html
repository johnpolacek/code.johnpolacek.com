The LightboxFormat class represents formatting information.

Use the LightboxFormat class to create specific formatting for Lightboxes.

You must use the constructor new LightboxFormat() to create a TextFormat object
before setting its properties. The Lightbox properties are set to default values
so if you don't provide values for the properties, Lightboxes use their own
default formatting.
