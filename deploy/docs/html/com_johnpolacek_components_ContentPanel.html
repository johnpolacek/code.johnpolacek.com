ContentPanel

Kind of class:public class
Package:com.johnpolacek.components
Inherits from:Sprite
Version:27 Apr 2010
Author:John Polacek, john@johnpolacek.com
Classpath:com.johnpolacek.components.ContentPanel
File last modified:Monday, 08 November 2010, 09:39:47
The ContentPanel class displays content in a panel
that can be formatted with a background and margins.

It is designed to work with the ContentContainer class
to provide an efficient method of loading and displaying
html-formatted content from
an xml file.
