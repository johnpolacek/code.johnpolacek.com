Scrollbox

Kind of class:public class
Package:com.johnpolacek.ui
Inherits from:Sprite

Author:John Polacek, john@johnpolacek.com
Classpath:com.johnpolacek.ui.Scrollbox
File last modified:Friday, 12 November 2010, 08:15:47
Simple scrollbar class for full screen flash apps
Example:

  • // create scrollbox for container sprite masked to 400 pixels high, 
// with a scrubber 40 pixels high with a color of 0x0099999 and a 
// track color of 0x003333 (100% opacity) with a width of 10 pixels
var scrollbox:Scrollbox = new Scrollbox(400, 0x009999, 1, 0x003333, 1, 10, 40);
scrollbox.setContent(container);
addChild(scrollbox);
