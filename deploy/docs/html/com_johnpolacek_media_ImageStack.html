ImageStack

Loads image files as ImageDisplay sprites and stacks them on top of each other.
Example:
  • import com.johnpolacek.media.ImageStack;
var slides:ImageStack = new ImageStack("slide1.jpg","slide2.jpg","slide3.jpg");
Events broadcasted to listeners:
  • Event.COMPLETE When load is complete.
