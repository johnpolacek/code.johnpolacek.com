ChromeContainer

Kind of class:public class
Package:com.johnpolacek.container
Inherits from:Sprite

Classpath:com.johnpolacek.container.ChromeContainer
File last modified:Friday, 12 November 2010, 08:14:26
The ChromeContainer class is used to display sprites
inside a RectangleGradientShape
Example:

  • Create a simple chrome button

    * 
    import flash.text.TextField;
import flash.text.TextFieldAutoSize;
import com.johnpolacek.container.ChromeContainer;

var tf:TextField = new TextField();
tf.autoSize = TextFieldAutoSize.LEFT;
tf.text = "Click Me!";
var buttonGraphic:ChromeContainer = new ChromeContainer();
buttonGraphic.paddingTop = buttonGraphic.paddingBottom = 5;
buttonGraphic.contain(tf);
addChild(buttonGraphic);
