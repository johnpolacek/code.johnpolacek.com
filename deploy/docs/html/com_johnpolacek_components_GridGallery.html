GridGallery

Loads an array of thumbnails and arranges them in a grid.
Clicking on a thumbnail launches content from a corresponding
array in a lightbox window.
Example:

  • Create grid gallery 8 images wide x 3 images high with 1 pixel spacing, then
    use build method to create - 1st param is array of thumbs, 2nd is array of content

    var gallery:GridGallery = new GridGallery(8, 3, 1); 
gallery.build(["thumb1.jpg","thumb2.jpg"...], ["image.jpg","video.flv"...]);
