SoundSpectrum

Kind of class:public class

Classpath:com.johnpolacek.media.SoundSpectrum

Sound spectrum visualizer animated within a rectangle container.
Example:
  • import com.johnpolacek.media.SoundSpectrum;
var soundSpectrum = new SoundSpectrum(400, 100, 0x333333, 1, 0x999999, .1);
addChild(soundSpectrum);
soundSpectrum.activate();
