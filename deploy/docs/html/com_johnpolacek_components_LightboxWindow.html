LightboxWindow

Kind of class:public class
Package:com.johnpolacek.components
Inherits from:Sprite
Author:John Polacek, john@johnpolacek.com
Classpath:com.johnpolacek.components.LightboxWindow
File last modified:Friday, 12 November 2010, 08:00:27
The LightboxWindow class contains the ContentDisplay sprite.
Uses the ContentDisplayCreator simple Factory to generate content containers
Events broadcasted to listeners:
  • new Event(LIGHTBOX_PREV) When prevButton is clicked.
  • new Event(LIGHTBOX_NEXT) When prevButton is clicked.
See also:
  • com.johnpolacek.display.ContentDisplayCreator
  • com.johnpolacek.display.ContentDisplay
