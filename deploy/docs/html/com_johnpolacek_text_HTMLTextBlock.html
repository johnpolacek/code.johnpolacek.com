The HTMLTextBlock class displays html text and manages
the use of StyleSheets and inline images
Example:

  • Create a text block containing the string "txt" that is
    500 pixels wide and is styled using the StyleSheet "ss":

    htmlTextBlock = new HTMLTextBlock(txt, 500, ss);

    Change the width of the HTMLTextBlock to 300:

    htmlTextBlock.setWidth(300);
