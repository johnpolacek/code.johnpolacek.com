SoundLoop

Loads audio files and plays them in a loop.
Sound playback begins automatically.
Example:
  • import com.johnpolacek.sound.SoundLoop;
var soundLoop:SoundLoop = new SoundLoop("sample1.mp3", "sample2.mp3", "sample3.mp3");
