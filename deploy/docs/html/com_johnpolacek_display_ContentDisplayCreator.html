A Simple Factory class that constructs ContentDisplay sprites
Uses the ContentCreator simple Factory to generate content containers
Example:
  • import com.johnpolacek.display.ContentDisplay;
import com.johnpolacek.display.ContentDisplayCreator;
var example:ContentDisplay;
var creator:ContentDisplayCreator = new ContentCreator();
example = creator.create("video.flv");
